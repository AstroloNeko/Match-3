# 订单货架三消原型

一个零依赖 H5 小游戏原型，用来验证“货架遮挡 + 暂存槽三消 + 顾客订单”的核心循环。

## 玩法

- 屏幕上方有 3 个订单，点击可选货物进入下方暂存槽。
- 暂存槽中三个同类货物会自动出货。
- 出货类型匹配订单时，会推进订单进度；订单完成后补新订单。
- 上层货物会遮挡周围下层货物，被挡住的货物不能选择。
- 每关完成目标订单数即可通关。
- 暂存槽满或倒计时结束则失败。
- “卡槽+1”按钮模拟激励视频广告奖励，每关默认 7 格，最多扩到 9 格。

## 难度曲线

关卡不再只是加一点数量，而是逐步引入新的压力：

- 第 1 关：教学关，小货架、少种类、普通单，基本让玩家熟悉规则。
- 第 2 关：难度急剧上升，货架直接扩到 8x9，急单、大单、双品类、冰冻货和捆绑货都会开始出现。
- 后续关卡：货架继续扩到 10x12，货物种类、目标订单数、货架密度和遮挡比例都会显著增加。
- 难度曲线更陡：复杂订单更早出现，急单耐心更短，高关卡补货更密、导演兜底更弱。
- 连续完成订单会给连单奖励；清出不匹配订单的散货会扣一点时间。

## 特殊货物

- 星标货：参与订单出货时给额外分数和时间。
- 冰冻货：进入暂存槽后需要经历若干次三消才会解冻，冻结期间占槽但不参与三消。
- 炸弹货：进入暂存槽后作为主动道具保留，拖到另一张卡上可以清除目标卡。
- 捆绑货：两个货物会显示紫色链条，点其中一个会把另一个也一起拉进暂存槽。

## 开始界面

游戏启动时会先显示开始界面，可以打开玩法教程查看订单、遮挡和特殊货物说明。
第 1 关通关后会出现“难度升级”过场，再进入断崖式加压的第 2 关。

## 可控随机

这版仍然不是完全随机：

- 开局会优先生成当前订单需要的货物。
- 完成订单后会围绕新订单补货。
- 如果当前可选货物无法服务订单或暂存槽中的对子，系统会按关卡难度决定是否暗中修正一个可选货物。
- 关卡越往后，导演系统越少兜底，玩家的卡槽管理会更重要。
